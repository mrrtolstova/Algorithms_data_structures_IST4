а) наибольшее,
b) наименьшее,
c) наиболее близкое к нулю.

- реализвовать создание случайного массива .
- реализовать нахождение наибольшего числа из имеющихся.
- реализовать нахождение наименьшего числа из имеющихся.
- реализовать нахождение наиболее близкого числа к нулю  из имеющихся.
