Задание данной лабораторной работы: из библиотеки слов отобрать все анограммы.
Данная работы была выполнена в программной среде Visual Studio на языке C++.
Основная логика работы прописана в файле Main.cpp
На какие части делится работа: 
1. Создание класса для чтения слов из файла (закончено 10.04.2020)
Для реализации данной задачи был создан отдельный файл Helpers.h, в котором реализовано чтение слов по строчно для дальнейшей работы с этими словами.
2. Нахождение анограмм (закончено 10.04.2020)
2.1. Создание класса Anagram 
Данный класс помогает разбирать слова и переводить каждый символ в число, чтобы потом их можно было отсортировать. После сортировки числа опять переводятся обратно в символы.
2.2 Создание класса InsertSort
Данный класс помогает сортировать данный массив слов с помощью сортировки вставками. 
2.3. Создание класса AnagramFounder
В данном классе будут создаваться объеты класса Anagram.
Для поиска анаграмм использует функцию Anagramm::SortedSymbols().     
Если отсортированный набор символов совпадает, то выводим слова используя функцию Anagramm::GetWord().
