Цель: 
Реализовать пирамидальную сортировку.

Задачи:

- реализовать обработку исключений ввода
- реализовать создание и рандомное заполнение массива
- реализовать функцию формирования "кучи"
- реализовать пирамидальную сортировку
 


Процесс:

+ реализация обработки исключений ввода (проверка на корректный ввод) (20.03.2020)
+ реализация создания и рандомного заполнения массива (20.03.2020) 
+ реализация алгоритма пирамидальной сортировки (20.03.2020)
+реализация формирования "кучи" (20.03.2020) 
+ реализация вывода массива (20.03.2020) 
 

Используемые функции:

• heapSort – функция, реализующая алгоритм пирамидальной сортировки;
• heapify - функция, реализующая формирование "кучи" из массива;
• main - функция, реализующая создание и заполнение массива, обработку ввода и вызов heapSort.
