
Задача: 

Реализовать пузырьковую сортировку.

Необходимо:

- реализвовать создание случайного массива
- реализовать пузырьковую сортировку по убыванию
- реализовать пузырьковую сортировку по возрастанию
- реализовать функцию вывода массива 

Процесс:

(23.03.2020) + реализована функция вывода массива
(23.03.2020) + реализована функция рандомизации чисел на интервале
(23.03.2020) + реализована функция создания массива, заполненного рандомными числами
(23.03.2020) + реализован алгоритм пузырьковой сортировки по убыванию
(23.03.2020) + реализован алгоритм пузырьковой сортировки по возрастанию

Реализованные функции:

• getRandomNumberInRange– Рандомизатор целых чисел;
• getRandomArray –Заполнение массива рандомными числами;
• DescendingBubbleSort – Пузырьковая сортировка по убыванию;
• AscendingBubbleSort – Пузырьковая сортировка по возрастанию;
• PrintArray – Вывод массива.
